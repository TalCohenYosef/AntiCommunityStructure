הרצת model.py ליצור את המודל עצמו
python level4/model.py

הקובץ הזה הוא המודל עצמו — הרשת העצבית שבה את משתמשת בשביל לחלק את הקודקודים לקבוצות.

בקצרה מה הוא עושה:

class AntiCommunityGNN(nn.Module)
מגדיר מודל GNN חדש לפרויקט שלך. הוא יורש מ־nn.Module, כלומר זה מודל PyTorch רגיל.

__init__(...)
פה המודל בונה את השכבות שלו:

conv1 — שכבת GCN ראשונה
conv2 — שכבת GCN שנייה
כלומר המודל עושה שני צעדים של העברת מידע בין שכנים בגרף.

forward(x, edge_index, edge_weight)
זה החישוב של המודל:

מקבל את הפיצ'רים של הקודקודים x
מעביר אותם דרך conv1
מפעיל ReLU
מעביר דרך conv2
מפעיל softmax
ובסוף מחזיר לכל קודקוד וקטור הסתברויות בגודל 2.

בפרויקט שלך המשמעות היא:

הקלט: גרף + פיצ'רים התחלתיים
הפלט: לכל קודקוד הסתברות להיות בקבוצה 0 או בקבוצה 1

כלומר אם קודם הכנת את הדאטה, והקובץ load_gnn_data.py טען אותו, אז model.py הוא החלק שבאמת מחשב את החלוקה.
